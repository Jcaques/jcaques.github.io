Static & Dynamic Routing Implementation

This project showcases static routing and dynamic routing implementation using OSPF and RIP.

Network Typology

Consists of the Following Devices:

  • 3 Routers
  • 3 Switches
  • 6 Devices (2 Per Switch)

The network is divided into 3 Lans:

  • LAN 1 with the network 192.168.10.0/24
  • LAN 2 with the network 192,168.20.0/24
  • LAN 3 with the network 192.168.30.0/24

The router for LAN 1 is R1 and it's connected to R2, LAN 2's router, via 192.168.1.0/30. From R2, LAN 2 connects to LAN 3 through R3 via 192.168.2.0/30.
